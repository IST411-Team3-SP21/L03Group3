# L03 Group 3 - NOAA Webservice

Before running the application you will need to download the GSON library, via maven


## From netbeans
- Right click on the project
- Click "Clean and Build"
- Wait to finish
- Run Application


## From IntelliJ
- Right click on the project
- Click "Maven"
- Click "Generate Sources and Update Folders"
- Wait to Finish  
- Run Application
